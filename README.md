# VoiceScribe - AI Based Speech to Text System for Indian Languages

## Project Overview
VoiceScribe is an AI-powered web application that converts speech to text for Indian languages in real time. Built as part of the Interdisciplinary Project at RNS Institute of Technology, Bengaluru.

## Features
- Real-time speech to text in 10 Indian languages (Kannada, Hindi, Tamil, Telugu, Malayalam, Marathi, Gujarati, Punjabi, Bengali, English)
- Translation to 18 global languages
- Auto language detection
- Text-to-speech for translated output
- Confidence score display
- Recording timer and word count
- Export transcription as PDF, Word, or TXT
- Share via WhatsApp and Email
- Transcription history with search
- Dark/Light mode toggle
- Upload audio file for transcription
- Mobile responsive design

## Tech Stack
- **Backend:** Python 3.14, Flask
- **Speech Recognition:** Google Speech API (via SpeechRecognition library)
- **Translation:** Google Translate (via deep-translator)
- **Language Detection:** langdetect
- **Frontend:** HTML5, CSS3, JavaScript (Web Speech API)
- **Export:** fpdf2 (PDF), python-docx (Word)
- **Version Control:** Git, GitHub
